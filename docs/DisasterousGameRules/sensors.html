SensorsÂ¶

Sensors enable you to predict disasters before they happen. Correctly guessing which disaster will happen next can allow you to choose the correct decree. There is a sensor for every disaster, and each one must be upgraded individually. The six sensors are named fire, tornado, blizzard, earthquake, monster, and ufo.

Weather can be difficult to predict. The likelihood that certain disasters will occur will not be completely accurate. You can upgrade your sensors by allocating effort to the sensor. Upgrading your sensor will make the sensor more accurate by decreasing the range up and down that it can be wrong.

Level Effort Cost Sensor Inaccuracy Range
Base 0 100
1 4000 50
2 8500 20
3 13500 1
